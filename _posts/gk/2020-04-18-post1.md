• ICMR (Indian Council of Medical Research) gave permission for the state of Uttar Pradesh to start pool testing of COVID-19. UP has become the first state to start pool testing. • Pool testing is being done in the state to increase the number of testing. With the number of cases increased to 558, the state expects there could be the state expects there could be more cases than the numbers provided. Hence pool testing has been adopted. • Pool Testing, if 10 samples of COVID-19 test negative, then it is an indicator that all the samples test negative. On the other hand, if the samples tsamples tested are not negative, individual testing will be done.
